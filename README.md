# Logitech-G19-s-Web-Browser
A Web Browser for your Logitech G19(s)

This Applet is designed to bring a Web Browser to your (still) beloved G19/G19s.

#Features
- Can display google

- JxBrowser (https://www.teamdev.com/jxbrowser)
- Java 7(or higher)
